# What are PIREPS?
PIREP stands for `Pilot Report`, basically a first hand account of something.

## $100 Hamburgers
A `$100 Hamburger` is what pilots say when they fly somewhere to get lunch. The $10 meal costs >$100 after fuel and plane costs.
